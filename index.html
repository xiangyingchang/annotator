开始一次标注会话

把 Markdown / HTML 内容粘贴到下方，或加载示例开始。选中文字即可加标注，标注会按顺序保留并可一键导出给 AI；AI 回应后可粘贴回来自动勾选完成。

提示：粘贴 HTML 源码（含标签），不是渲染后的页面。脚本不会被执行。
仅支持同源或开启了 Access-Control-Allow-Origin 的页面。如果加载失败，请在浏览器打开该页面后 Ctrl+U（查看源代码）→ Ctrl+A 全选 → 复制到「粘贴 HTML」标签页。

示例是一份产品需求文档片段（"智能书签功能 v1.2"），包含标题、列表、引用、加粗等常见结构，便于演示标注功能。

连接本地工作区后，工具会列出该目录下的 .md / .html / .htm / .txt 文件。点击文件名即可加载并开始标注。
需要 Chrome 86+ / Edge 86+（支持 File System Access API）。授权一次后目录名会被记住，下次不用重选。
